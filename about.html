About Us

In the grocery and retail industry, existing systems for dataset management often come with significant limitations. Users are typically restricted to uploading only PDF datasets, lack the capability for real-time graph generation based on user-defined variables, and do not offer predictive analytics. To address these gaps, we developed an innovative system that overcomes these constraints. Our platform supports various data formats, provides dynamic real-time visualizations, and delivers accurate predictions, enhancing decision-making processes. This comprehensive solution ensures that users can fully leverage their data for improved operational efficiency and strategic planning.

Methodology

Data Handling and Storage

Our system initiates with the user uploading a dataset, which is securely stored on our server and sent to a database for future use. Upon receiving questions related to the dataset, they are forwarded to our advanced Language Learning Model (LLM) for classification. The LLM determines whether the query pertains to predictions or summaries, facilitating appropriate downstream processes. This ensures efficient data management and question handling right from the onset.

Prediction Model Workflow

For prediction-related queries, the dataset is preprocessed on the server for optimal quality. The data is then converted into a 2D array data frame and sent to our real-time prediction model, which generates forecasts dynamically. Forecast accuracy is assessed using an error score or R² value. A score above 1 signifies high prediction accuracy, allowing the forecast to be displayed. If the score is below 0, a message indicates insufficient data for prediction, ensuring clear and reliable forecasting.

Graph Generation and Summarization

For graph-related queries, the LLM generates code based on user-defined variables. This code is processed by a graph generation function to create real-time visualizations, which are then sent to the server for frontend display. For summary-related questions, the LLM provides answers directly from the dataset, utilizing its advanced analytical capabilities to deliver concise and accurate summaries. This approach ensures comprehensive data analysis and visualization, effectively catering to diverse user needs.
