About our project

The purpose of this project is to design an end-to-end solution with the functionalities of predictive modeling, data visualization, and real-time analysis designed solely for grocery retailing. The system will enable importing and uploading files in the format of CSV and Excel that contains the data in tri-lingual language of English, Sinhala and Tamil. Thus, the above mentioned objectives of the project will facilitate to improve the decision making activity in the context of grocery retail industry.
